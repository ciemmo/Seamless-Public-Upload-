# Seamless-Public-Upload-
A public upload of my group project for my Software Engineering class at Sam Houston State University. The project is meant to be a mobile app that acts a virtual closet for users. Note that this repository is meant to publicly display the results of the entire group's work. I handled the database side of the project and had little to do with the front or backend. I, along with a group member by the name of Kris Compton, worked on all database-related files and code within the project. Everything else was handled by other members of the group.
